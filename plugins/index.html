# 自定义插件

相比自定义脚本，自定义插件(Plugin)是一种自定义程度更高的扩展方式。不仅可以监听Lyrebird事件，还可以自定义UI和接口。通过插件可以完成复杂的功能扩展,对Lyrebird能力进行扩展或者实现定制化的功能。
