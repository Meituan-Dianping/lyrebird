# 插件

相比检查器(Checker)，插件(Plugin)是一种自定义程度更高的扩展方式。不仅可以监听Lyrebird事件，还可以自定义UI和接口。通过插件可以完成复杂的功能扩展,对Lyrebird能力进行扩展或者实现定制化的功能。

# Android插件

支持Android设备的插件。为Lyrebird提供设备信息、日志、截图、命令执行、崩溃监控、APP信息及拉起应用功能。

# 安装

pip3 install lyrebird-android

# BugIt插件

将实时收集的设备信息、APP信息、请求记录、日志等作为描述或附件一键提交到BUG管理系统的工具。 除手动跳转到BugIt界面进行操作外也可通过检查器(Checker)触发的通知跳转到BugIt。

# 安装
